void JewelCollectorProject.Maps.Map.createMap ( int  dimension)

Método que cria uma matriz de Cell de acordo com a dimensão informada. No primeiro momento, gera a matriz com todos elementos do tipo Empty, que em seguida serão substituídos conforme a regra de validação:

  • Se a dimensão for igual a 10, insere os elementos em posições fixas na matriz.
  • Se a dimensão for diferente de 10, insere aleatoriamente os elementos realizando um cálculo proporcional para o máximo de cada elemento.
