Detailed Description

Classe criada para representar o robô no jogo. Possui os seguintes atributos: X e Y: Representam sua posição no jogo. Essa localização é usada como referência para as ações com os demais elementos do mapa. Bag e TotalScore: Representam o total de joias coletadas e os pontos acumulados com as coletas. Fuel: Representa o combustível do robô. PressedKeyStatus: Simboliza o comando direcionado ao robô durante o jogo. Quando por algum motivo ele não conseguir executar o comando, emitirá uma mensagem de aviso. Motor: Objeto do tipo Motor, utiliza o conceito de composição para simbolizar o motor do robô. Radar: Objeto do tipo Radar, utiliza o conceito de composição para simbolizar o radar que o robô possui.
